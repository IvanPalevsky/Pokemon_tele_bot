# Описание

## Тип моего проекта:
> Telegram-bot

## Функции и оссобенности:

> Функция /start:
- Начинает игру.

> Функция /go:
- Создает твоего покемона.

> Функция /attack_pok:
- Атакует покемона другого пользователя.

> Функция /info:
- Расскажет вам всю информацию о покемоне другого пользователя.
